## 👋 소개

저는 컴퓨터공학과 4학년으로, 프론트엔드 개발자를 목표로 공부하고 있습니다.
사용자가 “아 이거 편하다”라고 느끼는 순간을 만드는 데 가장 큰 흥미를 느낍니다.

2025 관광데이터공모전에서 야외방탈출 프로젝트로 장려상을 받았고,
개인적으로는 단주차(캠퍼스 주차 앱), DayFrame(커스텀 플래너 앱)을 설계하고 직접 구현했습니다.

최근에는 React 기반의 프론트엔드 개발과 UI/UX 개선에 집중해 학습을 이어가고 있습니다.으로 꾸준히 성장하며 UI/UX와 사용자 경험 개선에 집중하고 있습니다.

UI 설계 및 핵심 프론트엔드 화면 구현

API 구조 논의 및 기능 테스트 참여

###📅 DayFrame

사용자가 직접 구성하는 커스텀 플래너 앱

동적 컴포넌트 생성 및 캘린더 UX 구현

반응형 UI 및 레이아웃 설계

###🧭 NAvI(야외방탈출프로젝트)



관광 데이터를 활용한 위치 기반 방탈출 경험 설계

실제 국내 관광지를 배경으로 한 스토리 기반 야외 방탈출 콘텐츠 기획

Phaser를 활용해 맵 이동, 상호작용 오브젝트, 퍼즐 시스템 구현

GPS 기반 진행 흐름(장소 이동 → 힌트 해제 → 다음 스테이지) 구조 설계

모바일 환경에서도 자연스럽게 플레이할 수 있도록 UI 인터랙션 최적화

UI 프로토타입 및 상호작용 화면 제작
